## How to Run

1. Open the project in Visual Studio 2022.
2. Restore NuGet packages.
3. Run the project (F5).
4. Open two browser windows.
5. Place bids from one browser.
6. Watch the bids update in real time in the other browser.
7. Test automatic reconnection by stopping and restarting the server.

## Technologies Used

- ASP.NET Core
- WebSocket
- Entity Framework Core
- SQLite
